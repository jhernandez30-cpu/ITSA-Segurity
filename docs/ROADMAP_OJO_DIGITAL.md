# Roadmap Ojo Digital SaaS

## Fase 1: Landing + paginas de servicios

Consolidar Ojo Digital SaaS como producto principal, mantener paginas de servicios individuales y ordenar navegacion del sitio comercial.

## Fase 2: Plataforma demo con localStorage

Crear login, registro, dashboard, reportes, Excel dashboard, Power BI placeholder y admin con datos demo.

## Fase 3: Supabase Auth y base de datos real

Reemplazar localStorage por Supabase Auth, Postgres, tablas de usuarios, clientes, sitios, revisiones, reportes y roles.

## Fase 4: Motor real de monitoreo autorizado

Implementar revisiones de disponibilidad, SSL, SEO, rendimiento y seguridad basica, solo sobre activos propios o autorizados.

## Fase 5: Reportes PDF automaticos

Generar reportes mensuales por cliente con hallazgos, KPIs, recomendaciones y estado general.

## Fase 6: Integracion Power BI

Conectar Power BI a datos reales, configurar permisos, embed seguro y dashboards por cliente.

## Fase 7: Pagos y suscripciones

Integrar planes, facturacion, pagos y renovaciones.

## Fase 8: Alertas por correo/WhatsApp

Enviar alertas ante caidas, SSL vencido, degradacion de rendimiento o hallazgos criticos.

## Fase 9: App movil o PWA

Crear experiencia movil instalable para clientes y administradores.
